# CRM Management System

## Technologies

- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS
- JWT Authentication
- Mongoose

## Features

- User Registration
- User Login
- JWT Authentication
- Customer CRUD
- Protected Routes
- Responsive UI

## Installation

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Environment Variables

```
PORT=5000
MONGO_URI=your_connection_string
JWT_SECRET=your_secret
```
